DNA Base Encoding

DNA-based encryption uses the four nucleotide bases (A, T, G, C) to encode binary data. Each base pair can represent 2 bits of information, making it an efficient encoding mechanism. Our system extends this with quantum superposition to create quantum-resistant encryption.
